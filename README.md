# fly-ai-project-video-to-clip

## 폴더 설명
- "원본동영상" 폴더 : 엑셀 파일에 올려둔 아기 동영상(2~48번)을 모두 다운로드하여 모아둔 폴더
- "클립동영상" 폴더 : 원본동영상의 모든 영상들을 1초 간격의 클립으로 잘라 모아둔 폴더
    - 이름명 규칙 : {원본동영상 이름}-1, {원본동영상 이름}-2, ...
- "테스트용" 폴더 : 무시하세요
